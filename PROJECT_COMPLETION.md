## 🏗️ 技术架构

### 前端架构
```
Next.js 15 (App Router)
├── 认证层 (NextAuth.js)
├── UI 组件层 (Tailwind CSS + 自定义组件)
├── 状态管理 (React Hooks + Session)
└── API 通信层 (Fetch API)
```

### 后端架构
```
Next.js API Routes
├── 认证中间件
├── 数据验证层
├── 业务逻辑层
└── 数据访问层 (Prisma)
```

### 数据库架构
```
PostgreSQL (Supabase)
├── User (用户表)
├── Activity (活动表)  
├── Registration (报名表)
└── Comment (评论表)
```
